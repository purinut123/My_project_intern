# README

1.โหลดไฟล์ zip จาก GitHub แลกแตกไฟล์ลงเครื่อง

2.หลังจากแตกไฟล์แล้วทำการเปิดไฟล์ขึ้นมาโดยแยกเป็นสองส่วนคือ
 -Folder Front end, Back end สำหรับ VScode
 -ไฟล์ data base สำหรับ MySql

3.ทำการ run file data base เพื่อให้ระบบสามารถเข้าถึงและใช้งานข้อมูลในฐานข้อมูลได้

4.สร้างไฟล์. env ในไฟล์ folder sec2_gr4_ws_src แล้วทำการใส่ข้อมูลดังนี้
-DB_HOST = localhost
-DB_USER = root
-DB_NAME = sec2_gr4_database
-DB_PASS = {ใช้รหัสของตัวเอง}
-PORT = 8000

5.เปิด terminal ใน sec2_gr4_ws_src และลง node_modules โดยใช้คำสั่ง `npm install ` ใน terminal ของ folder sec2_gr4_ws_src

6.เปิดอีก terminal ใน sec_gr4_fe_src และทำเหมือนกับข้อ 5 แต่ทำใน folder sec_gr4_fe_src

7.ให้ทำการพิมพ์ npm start ในทั้งสอง Terminal

8.โดยเมื่อ npm start ในฝั่งของ front end หน้าเว็บจะเด้งให้ทันทีหากไม่เด้งให้พิมพ์ `http://localhost:3000/` ในแถบ URL เพื่อเป็นการเริ่มใช้งาน web 

9.ระบบจะทำการแสดงหน้า **Home Page** 

## Nav bar
ในทุกหน้าจะมี Nav bar ที่เชื่อมไปหน้าต่างๆแต่หน้าที่ใช้งานได้จะมี: Search, Contact us, Login

## Home Page
หน้านี้จะแสดงรายการของหนังสื่อโดยแบ่งเป็นส่วนๆอย่าง 10 อันดับแรกของหนังสือแต่ละประเภท เรื่องที่มาใหม่ รวมไปถึงเรื่องที่พักการเขียนอยู่

## Search Page 
สามารถเข้าหน้านี้ได้จากการกด search จาก nav bar โดยถ้ากรอกข้อมูลหรือเลือกหมวดหมู่และกด search จะเข้าไปหน้า search โดยมีค่าที่กรอกและเลือกไว้ด้วย

## Contact Page 
หน้านี้จะมีรูปภาพและชื่อของสมาชิกพร้อมกับลิ้งที่เชื่อมไปยังหน้า ig ของแต่ละคน

## Detail page 
หน้านี้จะแสดงรายละเอีนดของหนังสือที่เลือกจากทางหน้า home page หรือ ผลจากการ search 

## ฟังก์ชัน Login
1.	กดที่ **Login** ขวาบนของหน้าไหนก็ได้
2.	จากนั้นระบบจะเข้าไปยังหน้า **Login** โดยผู้ใข้จะต้องกรอกอีเมล และ password ที่ถูกต้อง
3.  เมื่อทำการเข้าสู่ระบบเสร็จสิ้น ระบบจะเข้าสู่หน้าหน้า Home โดย Nav bar ส่วน Login จะเปลี่ยนเป็น Profile

## ฟังก์ชัน Search 
1.  ในหน้า search ของเราจะมีช่องให้ผู้ใช้สามารถรอกชื่อของหนังสือที่ต้องการได้
2.  และผู้ใช้ยังสามารถเลือก filter อย่างประเภทของหนังสทือ สถานะ และความรุนแรงของเนื้อหา และยังสามารถเลือกให้การแสดงผลเรียงได้แบบต่างๆ


## User must be Admin to use these function
## ฟังก์ชัน Edit 
1.	ในแต่ละข้อมูลของ **User** หรือ **Book** จะมีปุ่ม **Edit** สำหรับการแก้ไขข้อมูล
2.	ให้กดที่ปุ่ม **Edit** เพื่อเข้าไปแก้ไขข้อมูลที่ต้องการเปลี่ยนแปลง
3.	กรอกข้อมูลที่ต้องการแก้ไข
4.	เมื่อแก้ไขข้อมูลเสร็จแล้ว ให้กดปุ่ม **update** ระบบจะทำการอัปเดตข้อมูลในระบบ

## ฟังก์ชัน Delete
1.	เมื่อกดปุ่ม **Delete** ระบบจะแสดงข้อความเพื่อยืนยันการลบข้อมูล โดยจะแสดงข้อความถามว่าจะลบข้อมูลนี้จริงๆ หรือไม่
2.	หากต้องการลบจริง ให้กดปุ่ม **Confirm** และข้อมูลจะถูกลบออกจากระบบ

## ฟังก์ชัน Add User
1.	เมื่อกดปุ่ม **Add user** หรือ **Add book** ระบบจะแสดงฟอร์มสำหรับการเพิ่มข้อมูล
2.	กรอกแบบฟอร์มให้ครับแล้วกด**save**
3.	ระบบจะทำการAdd ข้อมูล

## ฟังค์ชั่นการรีวิว
1. ผู้ใช้สามารถเพิ่มรีวิวให้หนังสือต่างๆได้จากการกดเลือกหนังสือจากหน้า home หรือผลการค้นหา
2. หังจากเข้ามา Detail เลือกเข้าไปหน้า Review
3. ถ้าผู้ใช้ไม่ได้เข้าสู่ระบบจะมีการ block ให้ไปเข้าสู่ระบบก่อน
4. เมื่อผู้ใช้เข้าสู่ระบบแล้วจะสามารถกรอกคะแนนได้ 1-5 และ เขียนรีวิวและยังสามารถใส่ gif ได้ด้วย
5. โดยหลังจากกดยืนยันรีวิวจะถุกบันทึก และแสดงผลหน้า Detail
